# Hướng dẫn Deploy Website với ShortLink.vn

## 1. Deploy lên GitHub Pages
1. Tạo repository trên GitHub
2. Upload tất cả files
3. Enable GitHub Pages
4. Lấy URL: https://username.github.io/repo-name

## 2. Tạo Shortlink tại shortlink.vn
1. Truy cập: https://shortlink.vn
2. Paste URL GitHub Pages
3. Custom alias: duongvu hoặc led-repair
4. Kết quả: https://shortlink.vn/duongvu

## 3. Marketing với Shortlink
- Business card: shortlink.vn/duongvu
- Facebook post: Dễ nhớ, gõ nhanh
- Zalo signature: Thêm vào chữ ký
- QR Code: shortlink.vn tự động tạo QR

## 4. Ưu điểm shortlink.vn
✅ Miễn phí
✅ Dịch vụ Việt Nam (tốc độ tốt)
✅ Có analytics cơ bản
✅ Tự động tạo QR code
✅ Không cần mua domain
✅ Setup nhanh (5 phút)

## 5. Theo dõi hiệu quả
- Click analytics
- Nguồn traffic
- Thời gian truy cập
- Thiết bị sử dụng
